Basic cookie 2

This page is to be viewed with the first page Basic cookie to show basic session cookies with no attributes. Basic cookies can be seen on multiple pages in the same path. The default path setting is for cookies to be able to be seen from pages in the same directory path.

You can clear the cookies on the previous page and this page and see how it works. The path sets to client which is the path of this page so any basic cookies created from pages in the client directory will share the same cookies.


Click the "Create Cookie" button.

This button adds the cookies:
document.cookie = common_cookie=from_page-2; cookie_2=from_page_2

Notice that adding a cookie adds it to the cookie bucket. If I was to add a cookie like this: document.cookie = another_cookie=another", it would add this cookie string on to the end of the cookies. It doesn't mean I have re-assigned document.cookies
