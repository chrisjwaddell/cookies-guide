Using <script> to create a cookie

I've included a <script> file that is used to send a cookie. This is how you can send the browser a cookie on page load if the website is hosted statically. Doing this on a website gives you an extra security check for things like login. If a request to login doesn't have this cookie, you can deny the request.

This webpage has the following line in the header:

<script src="http://localhost:25051/token"></script>

The Express code sends a cookie and a blank response.


router.get('/token', (req, res) => {
    res.set('Set-Cookie', 'from_script=script; path=/;')
    res.send("")
})
