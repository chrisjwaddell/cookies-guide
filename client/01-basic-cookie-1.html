Basic cookie

On this page, we create a basic cookie with no attributes. By default, cookies are session cookies.

Cookies need a key/value pair eg document.cookies = "name=Mark". You can create a cookie in the browser or in the server.

Create a basic cookie by clicking the Create Cookie button below.


Click the "Create Cookie" button.

This button adds the cookies:
document.cookie = common_cookie=from_page-1; cookie_1=from_page_1

This cookie is a basic one with no expiry date so when you close the browser and reopen it, this cookie will be deleted by the browser.

In the next page, you'll see that the cookie is also visible for that page as well. Cookies can be valid for some webpages and not valid for others.

On this page, we created cookie_1 with value - from_page_1 and on the next we'll create cookie_2 with value - from_page_2. And they'll both be visible.
