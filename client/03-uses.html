Uses and Advantages Cookies have

On this page, we create a basic cookie with no attributes. By default, cookies are session cookies.

Cookies need a key/value pair eg document.cookies = "name=Mark". You can create a cookie in the browser or in the server.

Create a basic cookie by clicking the Create Cookie button below.

When you close the page and reopen it, the cookies are deleted by the browser.

Note: You may need to close the browser and reopen it for it to delete the cookie because browsers often have session recovery built in. Not just the page tab but the whole browser.


Click the "Create Cookie" button.

This button adds the cookies:
document.cookie = common_cookie=from_page-1; cookie_1=from_page_1

Open the next page, cookie-02.html by clicking the link below.

Basic-Cookie-2.html

You can see that both pages can see common_cookie, it's value changes depending on which page you clicked Create Cookie on last.

cookie_1 with value - from_page_1 and cookie_2 with value - from_page_2 can be seen on both pages.
